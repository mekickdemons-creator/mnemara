# Contributors

Mnemara is built by humans working with AI, openly. We don't hide it
— we think it's the better workflow, and it's the *kind of workflow*
this project exists to support.

## Why this is in the repo

A few reasons:

1. **Honesty.** People should know how the project is built.
2. **Trust.** AI co-authorship is increasingly common in OSS; making
   it visible is better than pretending it didn't happen.
3. **It's on-thesis.** Mnemara is a runtime for working with LLMs.
   Hiding that LLMs helped build it would be incoherent.

If you contribute and would like to be listed, open a PR.
